html <applet> 标签

定义和用法

定义一个嵌入的 applet。

html 与 xhtml 之间的差异

在 html 4.01 中，applet 元素不被赞成使用。

在 xhtml 1.0 strict dtd 中，applet 元素不被支持。

提示和注释：

提示：如需包含 applet，请尽可能地使用 object 元素（因为在 xhtml 1.0 strict dtd 中，applet 元素不被支持）。

必需的属性

dtd 指示此属性允许在哪种 dtd 中使用。s=strict, t=transitional, f=frameset.

属性 值 描述 dtd
height pixels 定义 applet 的高度 tf
width pixels 定义 applet 的宽度 tf
