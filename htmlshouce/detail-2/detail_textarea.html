html <textarea> 标签

定义和用法

定义一个文本区域 (text-area) （一个多行的文本输入区域）。用户可在此文本区域中写文本。在一个文本区中，您可输入无限数量的文本。文本区中的默认字体是等宽字体 (fixed pitch)。

注释：在文本输入区内的文本行间，用 "%od%oa" （回车/换行）进行分隔。

提示：可以通过 <textarea> 标签的 wrap 属性设置文本输入区内的换行模式。有关 wrap 属性的详细信息。

html 与 xhtml 之间的差异

none

例子

<textarea rows="3" cols="30">
这里是文本域中的文本 ... ... ... ...
</textarea>

必需的属性

dtd 指示此属性允许在哪种 dtd 中使用。s=strict, t=transitional, f=frameset.

属性 值 描述 dtd
cols number 规定文本区内可见的列数。 stf
rows number 规定文本区内可见的行数。 stf

可选的属性

dtd指示此属性允许在哪种dtd中使用。s=strict, t=transitional, and f=frameset.

属性 值 描述 dtd
disabled disabled 当此文本区首次加载时禁用此文本区。 stf
name name_of_textarea 为此文本区规定一个名称。 stf
readonly readonly 指明用户无法修改文本区内的内容。 stf
