html <img> 标签

定义和用法

img 元素可定义一副图像。

html 与 xhtml 之间的差异

在 html 中，<img> 标签没有结束标签。

在 xhtml 中，<img> 标签必须被正确地关闭。

在 html 4.01 中，image 元素的"align"、"border"、"hspace" 以及 "vspace" 是不被赞成使用的。

在 xhtml 1.0 strict dtd 中，image 元素的 "align"、"border"、"hspace" 以及 "vspace" 是不被支持的。

mozilla firefox 和 alt 属性

如果您在 firefox 中使用 alt 属性，那么当鼠标悬浮在图片或图像地图时不会像其他的浏览器那样出现提示文本。

firefox 使用 alt 属性的方式与此属性被预想的方式是一致的，比如假如图像没有载入时出现的替代文本，而不是 mouse-over 文本。

例如，<img src="image.gif" alt="bla" title="bla bla bla" />

例子

<img src="angry.gif" alt="angry" />

必需的属性

dtd指示此属性允许在哪种dtd中使用。s=strict, t=transitional, f=frameset.

属性 值 描述 dtd
alt text 定义图像的简短描述（即图像的替代文本）。不超过 1024 个字符。 stf
src url 所显示图像的 url。 stf
