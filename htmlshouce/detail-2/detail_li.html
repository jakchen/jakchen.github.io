html <li> 标签

定义和用法

<li> 标签定义了列表项目。<li> 标签可用在有序列表 (<ol>) 和无序列表 (<ul>) 中。

html 与 xhtml 之间的差异

在 html 4.01 中，li 元素的 "type" 和 "value" 属性是不被赞成使用的。

在 xhtml 1.0 strict dtd 中，li 元素的 "type" 和 "value" 属性是不被支持的。

提示和注释：

注释：请使用样式来定义列表的类型。

例子

<ol>
   <li>coffee</li>
   <li>tea</li>
</ol>
<ul>
   <li>coffee</li>
   <li>tea</li>
</ul>
