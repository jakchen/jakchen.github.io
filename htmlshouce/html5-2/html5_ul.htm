HTML 5 <ul> 标签

定义和用法

<ul> 标签无序列表。

HTML 4.01 与 HTML 5 之间的差异

在 HTML 4.01 中，ul 元素的 "compact" 和 "type" 属性是不被赞成使用的。在 HTML 5 中，不再支持这两个属性。

提示和注释

提示：请使用 CSS 来定义列表的类型。

例子

<ul>
   <li>Coffee</li>
   <li>Tea</li>
</ul>
