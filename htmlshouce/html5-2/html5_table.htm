html 5 <table> 标签

定义和用法

<table> 标签可定义表格。在 <table> 标签内部，你可以放置表格的标题、表格行、表格列、表格单元以及其他的表格。

html 4.01 与 html 5 之间的差异

在 html 4.01 中，不赞成使用 "align" 和 "bgcolor" 属性。

在 html 5 中，不支持 <table> 标签的任何属性。

例子

<table>
<tr>
<td>cell a</td>
<td>cell b</td>
</tr>
</table>
