HTML 5 <nest> 标签

定义和用法

<nest> 标签在数据模板中为子元素定义嵌套点。与 <datatemplate> 和 <rule> 元素一同使用。

HTML 4.01 与 HTML 5 之间的差异

<nest> 标签是 HTML 5 中的新标签。

提示和注释

提示：插入的子节点必须匹配 <nest> 元素的 <rule> 元素。

属性

属性 值 描述
filter selector 定义能够进一步处理哪个子节点。
mode mode 定义数据模板的规则将拥有哪种模式。
