html 5 <datalist> 标签

定义和用法

<datalist> 标签定义可选数据的列表。与 input 元素配合使用，就可以制作出输入值的下拉列表。

html 4.01 与 html 5 之间的差异

<datalist> 标签是 html 5 中的新标签。

例子

<input id="mycar" list="cars" />
<datalist id="cars">
  <option value="bmw">
  <option value="ford">
  <option value="volvo">
</datalist>
