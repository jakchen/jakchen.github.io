HTML <legend> 标签

定义和用法

legend 元素可为 fieldset 定义一个标题。

实例

<fieldset>
<legend>Health information:</legend>
<form>
<label>Height: <input type="text" /></label>
<label>Weight: <input type="text" /></label>
</form>
</fieldset>

HTML 与 XHTML 之间的差异

在 HTML 4.01 中，align 属性不被赞成使用。

在 XHTML 1.0 Strict DTD 中，align 属性不被支持。

可选的属性

DTD 指示此属性允许在哪种 DTD 中使用。S=Strict, T=Transitional, F=Frameset.

属性 值 描述 DTD
align
  • top
  • bottom
  • left
  • right
 为 fieldset 中的内容定义排列方式。默认值是 Top。 TF
