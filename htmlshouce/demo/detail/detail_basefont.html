HTML <basefont> 标签

定义和用法

定义一个基准字体。

HTML 与 XHTML 之间的差异

在 HTML 4.01 中，basefont 元素是不被赞成使用的。

在 XHTML 1.0 Strict DTD 中，basefont 元素不被支持。
