HTML <address> 标签

定义和用法

<address> 可定义一个地址（比如电子邮件地址）。您应当使用它来定义地址、签名或者文档的作者身份。

HTML 与 XHTML 之间的差异

无。

提示和注释：

注释：address 通常被呈现为斜体。大多数浏览器会在 address 元素的前后添加一个换行符，不过如果有必要的话，您需要在地址文本的内容添加额外的换行符。
