HTML <area> 标签

定义和用法

定义图像映射中的某个区域。

HTML 与 XHTML 之间的差异

在 HTML 中，<area> 没有结束标签。

在 XHTML 中，<area> 必须被正确地关闭。

提示和注释：

此元素永远被嵌套在 <map> 标签内部。

注释：<img> 中的 usemap 属性可引用 <map> 中的 id 或 name 属性（由浏览器决定），所以我们需要同时向 <map> 添加 id 和 name 两个属性。
