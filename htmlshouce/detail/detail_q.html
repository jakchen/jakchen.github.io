HTML <q> 标签

定义和用法

<q> 标签可定义一个短的引用。

HTML 与 XHTML 之间的差异

NONE

提示和注释：

注释：根据 HTML 4.01 规范，q 元素应当使用分界引号来呈现，就就是说，q 元素包含的文本必须以引号来开始和结束。Mozilla 浏览器 (Firefox) 以及 Opera 符合这个规定，但是 Internet Explorer 却不支持此规定。

提示：<q> 标签在本质上与 <blockquote> 是一样的。不同之处在于它们的显示和应用。<q> 标签用于简短的行内引用。
