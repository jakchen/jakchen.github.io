HTML <dir> 标签

定义和用法

<dir> 标签可定义目录列表。

HTML 与 XHTML 之间的差异

在 HTML 4.01 中，dir 元素不被赞成使用。

在 XHTML 1.0 Strict DTD 中，dir 元素不被支持。

提示和注释：

注释：请使用 CSS 样式来定义列表的类型！

例子

<dir>
   <li>html</li>
   <li>xhtml</li>
</dir>
