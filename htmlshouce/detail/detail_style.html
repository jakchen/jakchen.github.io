HTML <style> 标签

定义和用法

style 元素位于 head 部分中。

如果您希望在页面中包含样式表，您应当在外部定义样式表，并使用 link 标签来链接它。

HTML 与 XHTML 之间的差异

NONE

提示和注释：

注释：如需学习更多有关样式表的知识，请访问我们的 CSS 教程。

例子

<head>
<style type="text/css">
h1 {color: red}
h2 {color: blue}
</style>
</head>
