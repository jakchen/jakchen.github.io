HTML 5 <hr> 标签

定义和用法

<hr> 标签水平线，它应该定义内容中的主题变化。

HTML 4.01 与 HTML 5 之间的差异

属性：在 HTML 4.01 中，不赞成使用 align, noshade, size 以及 width 属性。在 HTML 5 中不再支持这些属性。

在 HTML 5 中，<hr> 标签定义内容中的主题变化，并显示为一条水平线。

在 HTML 4.01 中，它仅仅显示为一条水平线。
