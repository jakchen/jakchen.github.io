HTML 5 <progress> 标签

定义和用法

<progress> 标签运行中的进程。可以使用 <progress> 标签来显示 JavaScript 中耗费时间的函数的进程。

HTML 4.01 与 HTML 5 之间的差异

<progress> 标签是 HTML 5 中的新标签。

提示和注释

提示：请使用 <progress> 标签来显示下载的进程。

例子

对象的下载进程：
<progress><span id="objprogress">85</span>%</progress>

属性

属性 值 描述
max number 定义完成的值。
value number 定义进程的当前值。
