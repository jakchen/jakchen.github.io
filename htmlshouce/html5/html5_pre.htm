HTML 5 <pre> 标签

定义和用法

pre 元素可定义预格式化的文本。被包围在 pre 元素中的文本通常会保留空格和换行符。而文本也会呈现为等宽字体。

<pre> 标签的一个常见应用就是用来表示计算机的源代码。

HTML 4.01 与 HTML 5 之间的差异

在 HTML 4.01 中，不赞成使用 width 属性，在 HTML 5 中不再支持该属性。

提示和注释

提示：请使用 <pre> 标签来显示非常规的格式化内容，或者某类计算机代码。
