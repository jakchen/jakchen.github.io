HTML 5 <source> 标签

定义和用法

<source> 标签为媒介元素（比如 <video> 和 <audio>）定义媒介资源。

HTML 4.01 与 HTML 5 之间的差异

<source> 标签是 HTML 5 中的新标签。

属性

属性 值 描述
media media query 定义媒介资源的类型，供浏览器决定是否下载。
src url 媒介的 URL。
type numeric value 定义播放器在音频流中的什么位置开始播放。默认地，音频从开头播放。
