HTML 5 <time> 标签

定义和用法

<time> 标签定义日期或时间，或者两者。

HTML 4.01 与 HTML 5 之间的差异

<time> 标签是 HTML 5 中的新标签。

例子

我们在每天早上 <time>9:00</time> 开始营业。
我在 <time datetime="2008-02-14">情人节</time> 有个约会。

属性

属性 值 描述
datetime datetime

定义元素的日期和时间。

如果未定义该属性，则必须在元素的内容中规定日期或时间。
