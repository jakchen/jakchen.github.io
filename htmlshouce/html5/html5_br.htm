HTML 5 <br> 标签

定义和用法

<br> 标签插入简单的换行符。

<br> 标签是一个空标签，意味着它没有结束标签。所以这样是错误的：<br></br>

HTML 4.01 与 HTML 5 之间的差异

无。

例子：

This text contains<br />a line break

输出：

This text contains
a line break
