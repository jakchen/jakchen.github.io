HTML 5 <nav> 标签

定义和用法

<nav> 标签定义导航链接的部分。

HTML 4.01 与 HTML 5 之间的差异

<nav> 标签是 HTML 5 中的新标签。

提示和注释

提示：如果文档中有“前后”按钮，则应该把它放到 <nav> 元素中。

例子

<nav>
  <a href="index.htm">mb5u 在线教程</a>
  <a href="html5_meter.htm">上一页</a>
  <a href="html5_nest.htm">下一页</a>
</nav>
