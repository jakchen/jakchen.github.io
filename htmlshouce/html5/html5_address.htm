HTML 5 <address> 标签

定义和用法

<address> 标签为文档或 section 定义联系信息。

HTML 4.01 与 HTML 5 之间的差异

NONE

例子：

<address>
此文档的作者：<a href="mailto:bill@microsoft.com">Bill Gates</a>
</address>

提示和注释：

注释：address 通常被呈现为斜体。大多数浏览器会在 address 元素的前后添加一个换行符，不过如果有必要的话，您需要在地址文本的内容添加额外的换行符。
