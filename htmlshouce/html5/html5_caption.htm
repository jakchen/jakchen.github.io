HTML 5 <caption> 标签

定义和用法

<caption> 标签定义表格标题。<caption> 标签必须紧随 <table> 标签之后。

HTML 4.01 与 HTML 5 之间的差异

HTML 4.01 不赞成使用 align 属性。在 HTML 5 中，不支持该标签。请使用 CSS 代替。

例子：

<table>
  <caption>This is a caption</caption>
  <tr>
  <td>Cell 1</td>
  <td>Cell 2</td>
  </tr>
</table>
