HTML 5 <button> 标签

定义和用法

<button> 标签定义按钮。您可以在 button 元素中放置内容，比如文档或图像。这是该元素与由 input 元素创建的按钮的不同之处。

HTML 4.01 与 HTML 5 之间的差异

在 HTML 5 中有一个新属性：autofocus。

例子：

<button>Click Me!</button>
